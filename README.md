# 📊 Dashboard de Vendas, Custos e Margem

## 📌 Sobre o Projeto
Este projeto tem como objetivo analisar dados de vendas, custos e margem de lucro, permitindo uma visão estratégica do desempenho financeiro.

## 🛠️ Ferramentas Utilizadas
- Power BI
- Excel

## 📊 Principais Análises
- KPIs de vendas, custo e margem
- Análise por categoria e região
- Identificação de oportunidades de melhoria
